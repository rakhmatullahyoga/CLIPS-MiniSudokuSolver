1.	Buka command prompt
2.	Masuk ke folder tempat ada .jar, lalu ketik "java -jar <nama file.jar>"
3.	Klik Sample 1, Sample 2, atau Sample 3 untuk meng-load soal dari spek
	Klik Custom untuk mengisi sendiri soal secara langsung pada board
	Klik Browse untuk meng-load soal dengna format CLIPS
4.	Klik Solve untuk menyelesaikan sudoku
5.	Klik Technique untuk melihat teknik apa yang digunakan dalam penyelesaian sudoku.
